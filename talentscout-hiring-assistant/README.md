# TalentScout Hiring Assistant

A simple AI-powered hiring assistant to help manage and evaluate candidates.

## Structure

- `app.py`: Main Streamlit application.
- `prompts/`: Contains prompt templates.
- `utils/`: Utility functions for LLM, storage, and conversation.
- `data/`: Storage for candidate data.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   streamlit run app.py
   ```
